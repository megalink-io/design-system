# Helpers

Helpers are small useful helper functions that perform a certain task and usually return a value as a result. They help to make the code of the application more understandable and clearer by outsourcing logic.

## getDeviceColorScheme

A helper function to get the prefers color scheme of the device.

## getDeviceType

A helper function to get the device type by viewport width.

## getHashCode

A helper function to get a hash code from a string, number or object.

## getRandomId

A helper function to get a random ID with uppercase, lowercase and numbers.

## getSearchParam

A helper function to get a URL search parameter by name.

## isValidColorCode

A helper function to check if a color code complies with the HEX, HEXA, RGB, RGBA, HSL or HSLA standard.
