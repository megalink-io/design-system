# Theme

A wrapper component that is necessary for the visual representation.

> The Theme component must be at the top of the component hierarchy. Since almost all components of the Design System depend on it.

## Key features

- Adds reset and default CSS code
- Adds font face CSS code for the font "Poppins"
- Updates the theme-color meta tag
- Adds or removes the "tabbing" CSS class
- Adds custom device type context and updates it when the window size changes
- Adds custom color scheme context
- Adds styled components theme context

## Code example

```tsx
import { Theme } from '@mds/components/basic/layout';
```

