# Dropdown

A drop-down field that allows users to select predefined values. Various decorations can be displayed in or around the field to communicate the entry requirements.

export const getProps = id => ({
  required: boolean('Required', true, id),
  disabled: boolean('Disabled', false, id),
});

```tsx

```

```tsx
<Dropdown
  name="example"
  value=""
  options={[
    {
      label: 'Option 1',
      value: 'option_1',
    },
    {
      label: 'Option 2',
      value: 'option_2',
    },
    {
      label: 'Option 3',
      value: 'option_3',
    },
  ]}
  error=""
  onChange={() => handleChange(name, value)}
  onBlur={() => handleBlur(name, value)}
  label="Options"
  placeholder="Select an option"
  required
/>
```
