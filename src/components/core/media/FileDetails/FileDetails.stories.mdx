# FileDetails

An overview of the details of a file. If possible, a preview of the file is displayed at the top.

<Preview withToolbar>
  <Story name="Default">
    <FileDetails file={file} />
  </Story>
</Preview>

> The component is optimized for use in our core. On this website the photos may therefore appear partially blurred.

## Code example

```tsx
import { FileDetails } from '@mds/components/core/media';
```

```tsx
<FileDetails file={file} />
```

## Props table

<Props of={FileDetails} />

> Currently only photos are supported. Other file formats may follow in the future.
