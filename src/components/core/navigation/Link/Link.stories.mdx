# Link

A navigation link that takes users to other content or allows them to download files. Contained text is underlined.

> In addition to simple text, the link component can also be applied to graphics or other elements.

<Preview withToolbar>
  <Story name="Internal link">
    <Link
      to={text('To', '/settings', 'internal_link')}
      replace={boolean('Replace', false, 'internal_link')}
      children={text('Children', 'Settings', 'internal_link')}
      underlined
    />
  </Story>
  <Story name="External link">
    <Link
      href={text('Href', 'https://www.example.com/privacy-policy', 'external_link')}
      target="_blank"
      children={text('Children', 'Privacy policy', 'external_link')}
      underlined
    />
  </Story>

    <Link

      children={

      }
    />
  </Story>
  <Story name="Download link">
    <Link
      href={text('Href', 'https://www.example.com/invoice.pdf', 'download_link')}

      children={text('Children', 'Download', 'download_link')}
      underlined

  </Story>

## Code example

```tsx
import { Link } from '@mds/components/core/navigation';
```

```tsx
<Link to="/settings">Settings</Link>
<Link href="https://www.example.com/privacy-policy" target="_blank" underlined>Privacy policy</Link>
```
